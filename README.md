# Getting started

## MacOS

Install Xcode from the App Store

```
# Install Xcode Command Line Tools
xcode-select --install
sudo xcodebuild -license accept

# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install git
brew install git

# Run script
cd macos
./install.sh
```
